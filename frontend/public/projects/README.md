# Project Images

Drop your project photos in this folder (`frontend/public/projects/`).

- Supported: .jpg, .jpeg, .png, .webp
- Reference them in code WITHOUT the word "public", e.g.:
    /projects/one-knock-1.jpg
- Suggested naming: <project-id>-1.jpg, <project-id>-2.jpg, ...
  e.g. one-knock-1.jpg ... one-knock-7.jpg

After you add files here, tell me the filenames (and which project)
and I'll wire them into the carousel in src/App.js.
